# AlphaForge Project Charter

## Vision

Build an enterprise-grade algorithmic trading platform that supports:

- Research
- Backtesting
- Paper Trading
- Live Trading
- Portfolio Analytics

## Goals

- Learn enterprise Angular architecture
- Learn NestJS
- Build a production-quality portfolio
- Support multiple brokers
- Support multiple trading strategies

## Non Goals

- Guaranteed profits
- High-frequency trading
- AI-based predictions in version 1

## Engineering Principles

- Clean Architecture
- SOLID
- Testable Code
- Small Commits
- Documentation First
